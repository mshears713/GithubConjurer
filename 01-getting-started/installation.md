# Installing GitHub Desktop

## Step-by-Step Installation Guide

### For Windows

1. **Download GitHub Desktop**
   - Go to: https://desktop.github.com/
   - Click the "Download for Windows" button
   - Save the installer file

2. **Run the Installer**
   - Double-click the downloaded file (GitHubDesktopSetup.exe)
   - The installer will automatically install the application
   - GitHub Desktop will launch when installation is complete

3. **Sign In**
   - When GitHub Desktop opens, you'll see a welcome screen
   - Click "Sign in to GitHub.com"
   - Enter your GitHub username and password
   - (If you don't have a GitHub account, click "Create your free account")

4. **Authorize GitHub Desktop**
   - Your browser will open
   - Click "Authorize Desktop"
   - Return to GitHub Desktop

### For Mac

1. **Download GitHub Desktop**
   - Go to: https://desktop.github.com/
   - Click the "Download for macOS" button
   - Save the .zip file

2. **Install the Application**
   - Open your Downloads folder
   - Double-click the .zip file to extract it
   - Drag GitHub Desktop to your Applications folder
   - Open GitHub Desktop from Applications

3. **Sign In**
   - When GitHub Desktop opens, you'll see a welcome screen
   - Click "Sign in to GitHub.com"
   - Enter your GitHub username and password
   - (If you don't have a GitHub account, click "Create your free account")

4. **Authorize GitHub Desktop**
   - Your browser will open
   - Click "Authorize Desktop"
   - Return to GitHub Desktop

### Creating a GitHub Account (If Needed)

If you don't have a GitHub account:

1. Go to https://github.com/join
2. Enter your email address
3. Create a password
4. Choose a username
5. Verify you're not a robot
6. Click "Create account"
7. Verify your email address

**Username Tips**:
- Choose something professional (you might use this for job applications!)
- Keep it simple and memorable
- You can use your real name or a professional handle
- Examples: john-smith, jsmith-dev, smithcodes

## System Requirements

### Windows
- Windows 10 64-bit or later
- 500 MB available disk space

### Mac
- macOS 10.13 or later
- 500 MB available disk space

## Troubleshooting Installation

### Windows Issues

**Problem**: "Windows protected your PC" message
- **Solution**: Click "More info" then "Run anyway"

**Problem**: Installation fails
- **Solution**: Make sure you have administrator rights
- Try right-clicking the installer and selecting "Run as administrator"

### Mac Issues

**Problem**: "GitHub Desktop can't be opened because it is from an unidentified developer"
- **Solution**:
  1. Go to System Preferences > Security & Privacy
  2. Click "Open Anyway"
  3. Confirm by clicking "Open"

**Problem**: Can't move to Applications folder
- **Solution**: Make sure you have permission to modify Applications folder

## What's Installed?

GitHub Desktop includes:

- **The GitHub Desktop app** - The main application
- **Git** - The version control system (installed automatically)
- **Command line tools** - For advanced users (optional)

## Alternative: Using GitHub Desktop Without an Account

You can use GitHub Desktop without a GitHub.com account, but you'll miss out on:
- Backing up your code to the cloud
- Collaborating with others
- Showcasing your projects publicly

**Recommendation**: Create a free GitHub account - it's worth it!
